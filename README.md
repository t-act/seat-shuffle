# 席替え (seating-change)

TS + React + Vite で作ったブラウザ完結の席替えツール。

## 機能
- 行×列のグリッドで席配置を指定（一部マスを「席なし」にできる）
- 名前の一括入力 / 1人ずつ編集
- 人単位で固定席を指定可能
- 基本ランダムで割り振り、シャッフル時は簡易アニメーション
- 席数と人数が合わない場合は警告を表示
- 結果を PNG 画像として保存

## 起動
```
npm install
npm run dev
```
ブラウザで http://localhost:5173 を開く。

## ビルド
```
npm run build
```

## 実装メモ
- 永続化は行わない（リロードで状態はリセット）
- 配置ロジックは `src/lib/shuffle.ts` の純粋関数
- 型定義は `src/lib/types.ts`
